## 📌 Features
- 🔍 Network Scanning (Nmap - 5 scan types)
- 🛡️ Vulnerability Assessment (CVE-based)
- 🌐 Web App Testing (SQLi, XSS, Directory Traversal)
- 📊 Auto Report Generation (TXT + JSON)
- 📝 Session Logging

- Python 3
- Nmap
- Kali Linux / Windows
- Burp Suite (complementary)
- Metasploit (complementary)
